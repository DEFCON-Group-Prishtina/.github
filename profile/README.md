# DEF CON Group Prishtina (DCG 38338)

website: https://dc38338.org

DEF CON Group Prishtina is the official local DCG chapter for Prishtina, Kosovo. Our focus is **hacker culture**.

We promote curiosity, experimentation, creative use of technology, learning by disassembling, and sharing knowledge openly and responsibly.We aim to connect the local hacker scene in Kosovo with the wider global DEF CON community.
